# 방탈출을 하고싶어요 · 리뷰 아카이브

[@want_escape_](https://www.instagram.com/want_escape_/) 가 직접 플레이한 방탈출 테마 리뷰를 한 곳에 모아 탐색할 수 있는 웹 아카이브입니다.

**바로가기 → https://want-escape.vercel.app/**

---

## 이 사이트는 어떤 곳인가요?

- 채널 운영자가 **직접 플레이한 방탈출 테마**만 등록되어 있습니다.
- 모든 평가(추천도·세부 평점·한줄평·탈출 팁)는 운영자의 **주관적인 리뷰**입니다.
- 원본 데이터는 구글 시트로 관리되며, 인스타그램 원문 리뷰로도 바로 이동할 수 있습니다.

---

## 사용 가이드

### 1. 홈 화면 — 테마 탐색

접속하면 운영중인 테마 카드가 **추천도 순**으로 정렬되어 보입니다. 상단의 검색창과 필터로 원하는 테마를 좁혀보세요.

#### 🔍 검색
테마 이름, 지점(카페)명, 지역, 한줄평 등을 한꺼번에 검색할 수 있습니다.
> 예: `홍대`, `공포`, `키이스케이프`, `감성`

#### 🧭 필터
| 필터 | 설명 |
|------|------|
| **지역** | 시/도 또는 권역 단위로 테마를 필터링합니다. |
| **최소 추천도** | 선택한 등급 이상의 테마만 표시합니다. (예: `A 이상`) |
| **정렬** | 추천도 순 / 지역 순 / 난이도 낮은 순 / 난이도 높은 순 |
| **폐업 포함** | 기본값은 운영중 테마만. 토글하면 폐업·기간한정 종료 테마도 함께 보여줍니다. |
| **초기화** | 적용한 모든 검색어와 필터를 한 번에 해제합니다. |

#### 📊 추천도 등급 안내
최상위부터 아래로 내려갈수록 선호도가 낮아집니다.

`S++` · `S+` · `S` · `A+` · `A` · `B+` · `B` · `C+` · `C` · `F` · `X` · `Misc`

---

### 2. 테마 상세 화면

카드를 클릭하면 상세 페이지로 이동합니다.

- **기본 정보** — 추천도, 난이도, 공포도, 활동성
- **세부 평점** — 문제 / 인테리어 / 스토리 / 창의성 / 연출 (5점 만점, 간혹 6점 이상의 예외적 평점 포함)
- **한줄평** — 테마의 인상을 압축한 코멘트
- **탈출 팁** — 스포일러 없는 선에서 도움이 될 힌트
- **비고** — 기간 한정, 이벤트 등 특수 운영 정보
- **원문 리뷰** — 인스타그램 임베드 또는 원문 게시물 링크
- **관련 테마 리뷰** — 같은 카페 / 비슷한 난이도·공포도·활동성의 다른 테마 추천

하단의 **더보기** 버튼으로 관련 테마를 더 불러올 수 있고, 좌상단의 **전체 목록** 링크로 홈으로 돌아갑니다.

---

### 3. 다크 모드

브라우저(또는 OS)의 **다크 모드 설정을 자동으로 따라갑니다.** 별도의 토글 없이도 쾌적하게 볼 수 있어요. 한 번 선택한 테마 설정은 로컬에 저장되어 재방문 시에도 유지됩니다.

---

### 4. 데이터 최신화

- 첫 방문 시 구글 시트에서 최신 데이터를 불러옵니다.
- 이후에는 **1시간 동안은 캐시**된 데이터를 즉시 보여주고, 그 이후 방문에서는 백그라운드에서 자동으로 새 데이터를 가져옵니다.
- 강제로 새로고침하려면 브라우저 새로고침(Cmd/Ctrl + Shift + R) 또는 로컬 스토리지의 `we-dataset-v1` 항목을 지우면 됩니다.
- 히어로 영역의 "최신 업데이트" 날짜로 시트 기준 반영일을 확인할 수 있습니다.

## 자주 묻는 질문

**Q. 평가 기준이 궁금해요.**
모두 운영자의 주관적 평가입니다. 절대 기준이 아니라 “이 리뷰어의 취향 기준”이라고 생각하고 봐주세요.

**Q. 데이터 로드 오류가 떠요.**
구글 시트에서 직접 파일을 받아오는 구조라, 네트워크 상황이나 브라우저 차단 정책(CORS 등)에 따라 실패할 수 있습니다. 새로고침을 하거나 다른 네트워크에서 재시도해 보세요.

**Q. 테마 등록/수정 요청은 어디로?**
인스타그램 [@want_escape_](https://www.instagram.com/want_escape_/) DM으로 연락해 주세요.

**Q. 폐업한 테마는 왜 기본적으로 숨겨지나요?**
현재 플레이 가능한 테마를 빠르게 찾기 위해서입니다. 기록 목적으로 보고 싶다면 **"폐업 포함"** 토글을 켜주세요.
